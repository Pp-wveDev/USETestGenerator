# USETestGenerator
Herramienta creada como trabajo de fin de grado en la Universidad de Málaga.

## ¿Qué hace?
Esta herramienta es capaz de generar casos de prueba de forma automática a partir de un modelo generado en USE, basándose en las pre y post condiciones e invariantes de este.
